# 📊 Startup Funding Analysis Dashboard

## 📌 Project Overview

This project analyzes startup funding trends using an interactive dashboard built in **Power BI**.
The dashboard helps understand how startup investments vary across industries, cities, and years.

It provides insights into the most funded startups, investment trends over time, and funding distribution across different sectors.

---

## 🛠 Tools & Technologies

* Microsoft Power BI
* Microsoft Excel
* Data Visualization
* Data Analysis

---

## 📂 Dataset Information

The dataset contains information about startup funding including:

* Startup Name
* Industry
* City
* Country
* Funding Stage
* Funding Amount ($)
* Investors
* Date

Total records in dataset: **65**

---

## 📊 Dashboard Features

The dashboard includes the following visualizations:

* **Total Funding KPI** – Displays total investment amount.
* **Total Startups KPI** – Shows number of startups in dataset.
* **Average Funding KPI** – Shows average investment per startup.
* **Funding by Industry** – Bar chart showing funding distribution across industries.
* **Funding by City** – Map visualization showing funding across Indian cities.
* **Top 10 Funded Startups** – Highlights startups with highest funding.
* **Funding Trend Over Time** – Area chart showing yearly funding growth.
* **Interactive Filters (Slicers)**:

  * Country
  * Industry
  * Funding Stage
  * Year

---

## 📈 Key Insights

* E-commerce and FinTech industries receive the highest startup investments.
* Major startup investments are concentrated in cities like **Bangalore, Mumbai, and Delhi**.
* Startup funding shows a steady increase over the years.
* Series A and Series B funding stages dominate the investment landscape.

---


## 🚀 How to Use the Project

1. Download the `.pbix` file from this repository.
2. Open it using **Power BI Desktop**.
3. Interact with the dashboard using filters and slicers.
4. Explore insights from different visualizations.
